# UTF-8 Visualizer

A tool to visualize how Unicode code points get encoded into UTF-8.

Code repository mirrors: [GitHub](https://github.com/ardislu/utf-8-visualizer), [Codeberg](https://codeberg.org/ardislu/utf-8-visualizer), [git.ardis.lu](https://git.ardis.lu/utf-8-visualizer)

## Local development

Use any local web server to host the static files. For example, if you have Node.js installed:

1. Install a local web server:

```
npm i -g local-web-server
```

2. Serve the files:

```
ws
```
